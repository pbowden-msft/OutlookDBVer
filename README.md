# OutlookDBVer
<b>Microsoft Outlook 365/2019/2016 Database Version</b>

Purpose: Reports the version of the Outlook 365/2019/2016 database</br>
Usage: `OutlookDBVer [--DefaultDB] [--Path:<path_to_Outlook.sqlite>]`</br>
